Esencia de la Marca

Preview AI ayuda a los usuarios a ver antes de comprar, integrándose en la PDP como si fuera una función nativa, pero transmitiendo que hay tecnología avanzada detrás.

Tono visual: limpio, neutro, profesional.

Personalidad: inteligente, útil, transparente.

Objetivo: no robar protagonismo al producto, sino realzar la confianza del comprador.

2. Paleta de Colores

La paleta se basa en WooCommerce y WordPress, con sutiles acentos para comunicar IA sin sobresaturar.

| Propósito                          | Color              | Hex       |
|------------------------------------|--------------------|-----------|
| **Primary**: marca + CTAs          | Electric Blue      | `#3B82F6` |
| **Secondary**: acentos secundarios | Soft Purple        | `#7C3AED` |
| Neutral oscuro: textos             | Charcoal           | `#1F1F1F` |
| Neutral medio: bordes / UI         | Slate Gray         | `#D7DDE4` |
| Neutral claro: backgrounds limpios | Off-White          | `#F7F8FA` |
| Éxito (generado)                   | Mint Green         | `#3DD5A2` |
| Error / invalidación               | Soft Red           | `#E76A6A` |

Regla: El azul primario se usa en acciones principales (botones CTA). El morado secundario para acentos y estados hover cuando sea apropiado.

3. Tipografía

Alineada al ecosistema WordPress/WooCommerce.

Fuente: Inter (preferente) o system-ui

Inter 500 → Botones y labels

Inter 400 → Texto general

Inter 300 → Mensajes secundarios

Características:

Altamente legible

Minimalista

No ornamental

4. Iconografía

Estilo outlined, líneas finas (1.5px), esquinas ligeramente redondeadas.

Iconos esenciales:

Upload (arrow-up-tray)

IA/processing (sparkle o stars)

Resultado (check-circle)

Error (exclamation-circle)

Cambiar foto (arrow-path)

Regla: Iconos deben verse como parte del UI nativo, no caricaturescos.

5. Componentes del Widget Preview AI
5.1 Contenedor principal

Fondo: #FFFFFF o transparente según tema

Borde: 1px solid #D7DDE4

Radio: 6px

Padding: 16px

Shadow (opcional): 0 2px 6px rgba(0,0,0,0.04)

El widget debe fundirse con la PDP: no parecer un pop-up invasivo.
